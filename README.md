 Desarrollo del Proyecto – Tienda de Ropa Web

En este proyecto desarrollamos una página web de una tienda de ropa que simula el funcionamiento de un comercio online. La página permite que los usuarios se registren, inicien sesión, visualicen productos con sus respectivas imágenes, los agreguen al carrito y realicen pedidos, logrando una experiencia clara y organizada.

El proyecto está estructurado dentro de una carpeta principal llamada tienda_ropa, donde organizamos las distintas partes para mantener todo ordenado. Allí se encuentran las páginas de la web, la parte interna del funcionamiento y la sección destinada a las imágenes de los productos. Esto nos permitió trabajar de manera más prolija y que cada elemento tenga su lugar dentro del sistema.

A lo largo del desarrollo incorporamos validaciones en toda la página para controlar los datos ingresados por los usuarios, evitando errores y asegurando que la información sea correcta. También desarrollamos la parte de administrador, desde donde se pueden gestionar productos, usuarios y pedidos, lo que completa el funcionamiento general de la tienda.

Nos dividimos las tareas de forma equilibrada, aunque trabajamos constantemente en conjunto para integrar cada parte y asegurarnos de que todo funcione correctamente.

 Desarrollo

El desarrollo lo realizamos utilizando Visual Studio Code y, una vez finalizado, subimos el proyecto a GitHub para poder compartirlo.

 Conclusión

Este proyecto representó un desafío importante para nosotros, ya que implicó integrar muchas partes distintas dentro de una misma aplicación. Durante el desarrollo tuvimos varias complicaciones, especialmente al momento de conectar las diferentes secciones del sistema y lograr que todo funcione de manera coherente. En varias ocasiones surgieron errores que nos llevaron tiempo de prueba y corrección hasta poder resolverlos.

En algunos puntos del proyecto también tuvimos que utilizar herramientas de inteligencia artificial para poder avanzar, principalmente cuando necesitábamos entender ciertos errores o encontrar una forma de encarar alguna funcionalidad que no nos estaba saliendo. La utilizamos como apoyo y guía, analizando siempre las respuestas para comprender lo que estábamos implementando y adaptándolo a nuestro trabajo.

En general, este proyecto nos permitió aplicar los conocimientos vistos en la materia, mejorar nuestra capacidad de resolución de problemas y fortalecer el trabajo en equipo. Más allá de las dificultades que surgieron en el proceso, logramos completar una aplicación funcional y entender de manera más práctica cómo se desarrolla una página web completa.
